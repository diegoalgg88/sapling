# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-03-02

### Added

- Initial release
- Agent loop with inter-turn context management
- Tool system: bash, read, write, edit, grep, glob
- LLM backends: Claude Code subprocess (cc), Anthropic SDK (sdk)
- Context manager pipeline: measure, score, prune, archive, reshape
- CLI: `sapling run` command with model/backend/cwd options
